Find triplets with zero sum
Difficulty: MediumAccuracy: 25.81%Submissions: 348K+Points: 4Average Time: 20m
Given an array arr[] of integers, determine whether it contains a triplet whose sum equals zero. Return true if such a triplet exists, otherwise, return false.

Examples:

Input: arr[] = [0, -1, 2, -3, 1]
Output: true
Explanation: The triplet [0, -1, 1] has a sum equal to zero.
Input: arr[] = [1, 2, 3]
Output: false
Explanation: No triplet with a sum of zero exists.
Input: arr[] = [-5, 3, 2, -1, 0, 1]
Output: true
Explanation: The triplet [-5, 3, 2] has a sum equal to zero.
Constraints:
1 ≤ arr.size() ≤ 103
-106 ≤ arr[i] ≤ 106


class Solution {
    public boolean findTriplets(int[] arr) {
        // code here.
        
        if(arr.length<3)
            return false;
            
        Arrays.sort(arr);
        
        for(int i=0;i<arr.length;i++){
            int low = i+1;
            int high = arr.length-1;
            
            while(low<high){
                int sum = arr[i] + arr[low] + arr[high];
                
                if(sum<0){
                    low++;
                }else if(sum>0){
                    high--;
                }else{
                    return true;
                }
            }
        }
        return false;
    }
}
